echo:echo

Full name:

com.soebes.maven.plugins:echo-maven-plugin:0.5.0-SNAPSHOT:echo

Description:

(no description)

Attributes:

  • Requires a Maven project to be executed.

Required Parameters

Name Type Since Description
<echos> List - The messages you would like to print out. 
   ..
   <configuration>
     <echos>
       <echo>Message Line Nr. 1</echo>
       <echo>Message Line Nr. 2</echo>
     </echos>
   </configuration>
   ..

<logLevel> EchoMojo$LogLevels - This will define in which logging level the given messages will be printed out. Their exist the following logging levels:
  • INFO
  • WARNING
  • ERROR
  • DEBUG

Default value is: INFO.

Optional Parameters

Name Type Since Description
<verbose> boolean - This will turn on verbose behavior and will print out all information.
Default value is: false.
